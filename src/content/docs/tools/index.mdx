# JMeter Tools

Interactive helpers for common load-testing decisions. All tools run **in your browser** on [docs.jmeter.ai](https://docs.jmeter.ai): no account, no upload of test data.

## Available tools

| Tool | Use it when you need to… |
|------|--------------------------|
| [Thread Calculator](/tools/thread-calculator/) | Estimate concurrent threads / RPS and ramp-up |
| [Heap Estimator](/tools/heap-estimator/) | Pick a starting `-Xmx` for injectors |
| [Properties Cheat Sheet](/tools/properties-cheatsheet/) | Find high-impact `user.properties` quickly |
| [Regex Extractor Builder](/tools/regex-tester/) | Paste a response (max 1 MB, browser-only) for Regex Extractor fields |

## Why these tools

Docs alone rank for "what is…", but practitioners search for calculators and cheat sheets. Each tool has shareable URL state, SEO metadata, and links back into the manual so answers stay grounded in JMeter concepts.
